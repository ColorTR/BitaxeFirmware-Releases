# BitaxeFirmware-Releases

Public release channel for ColorTR Bitaxe firmware binaries.

## Policy

- This repository is a release-asset channel, not a development repo.
- Final downloadable files are published as GitHub Release assets.
- The branch itself should stay lightweight and not carry versioned binary drops.
- The canonical public asset pair is:
  - `esp-miner.bin`
  - `www.bin`

## Notes

- Transitional CI recovery releases may be superseded and removed from the public release list once their fixes are absorbed into the next clean public version.
- Source-of-truth code remains in the main firmware repository.
